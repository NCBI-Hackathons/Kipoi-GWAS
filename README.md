## Installation

1. Clone the repo.

```
git clone https://github.com/NCBI-Hackathons/Kipoi-GWAS
```

2. Install conda environment.

```
conda env create -f environment.yml
```

3. Install the python package
```
cd Kipoi-GWAS; pip install .
```
